Spiral toolkit includes set of virtual tags, frontend library and resource manager.

Elements included
-----------------
* ajax forms
    * simple input
    * selectbox
    * switch
    * textarea
    * radios and check boxes
* paginator element
* lorem generator
* simple grid element with paginator
    * simple cell
    * numeric cell
    * boolean cell
    * bytes (size) cell
* caching element
* asset importing elements
    * css file
    * js file
    * inline js
